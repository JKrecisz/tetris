## Installation

Firstly, install pygame module: 

```bash
 pip3 install pygame
 # or
 pip install pygame
```

Then download the game, and cd to them:

```bash
 git clone https://github.com/JKrecisz/tetris.git
 cd tetris
```

Finally run and enjoy!:

```bash
 python3 tetris_main.py
 # or
 python tetris_main.py
```
